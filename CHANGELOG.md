# Changelog

All changes to Normalize including new features, updates, and removals are documented here.


## [v0.0.8] - 2026-07-22

### Distribution Files
- **Mini file**: https://unpkg.com/@codady/normalize@0.0.8/dist/normalize.min.css
- **Taiwind file**: https://unpkg.com/@codady/normalize@0.0.8/dist/normalize.tailwind.min.css

### Changes

#### Fixed
- Set `img`, `audio`, and `video` tags as block elements.将img、audio和video标签设为block元素。

#### Added
- Add `[hidden]:where(:not([hidden='until-found']))`. 新增`[hidden]:where(:not([hidden='until-found']))`。

#### Removed
- Remove the `text-decoration: underline dotted` property from the `abbr` tag.移除abbr标签的`text-decoration: underline dotted`属性
