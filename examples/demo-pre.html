🧪 pre / code 标签演示

① 单独使用 <code>

这是一个行内代码示例：const x = 10;

适用于：变量名、函数名、简短代码片段。

② 单独使用 <pre>

function hello() {
    console.log("Hello World");
}

特点：保留空格和换行，但语义上只是“预格式文本”。

③ <pre> 嵌套 <code>（推荐）

function hello() {
  console.log("Hello World");
}

✅ 这是标准写法，语义更清晰：
- <pre>：负责排版（保留格式）
- <code>：表示“这是代码”

④ 对比总结

  • <code>：行内代码（不会换行）
  • <pre>：保留格式文本
  • <pre><code>：最佳实践（代码块）
